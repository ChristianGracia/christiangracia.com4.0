Dockerized

This site is deployed in a docker container where this application is built in a NginX web server

Responsive Global Fonts

This site has no font-size declarations in the entire application besides pre-defined font sizes in my custom Angular Material theme. I made these fonts shrink or grow depending on screen size while most sites I see use media queries which causes font-sizes to not shrink or grow but a noticeable gap up or down when a certain screen size is reached.

Lazy Loaded Modules

Each page of this website is lazy loaded as a single unit when it is requested. So when you access the home page, you haven't loaded the about or project page yet but you reached the home page faster then you would have if you loaded all the modules

Spotify API Integrated

This website receives my Spotify listening data through my API server. Every 24 hours this server logs into Spotify with a Google Chrome bot to get an authorizaion code which is used to get an access token. This access token is then refreshed every 55 minutes throughout the day to provide the Spotify data shown on my website. Below, received from my API back-end, is my class file Spotify.ts which contains all this logic.

Web Analytics

When someone visits my website, a request is made to my API that emails me collected user data.

Full Material Theming

This site fully used Material theming to provide a uniform look and use less code + seamless Light/Dark mode
